# Aplikasi PUSKESMAS Sederhana

Aplikasi <b>PUSKESMAS</b> sederhana berbasis desktop, dibuat mengunakan <b>VISUAL STUDIO 2010</b> dengan bahasa pemrograman <b>VB.NET</b> dan database SQL SERVER 2008.
